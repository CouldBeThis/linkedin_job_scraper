# Linkedin Jobs Scraper

Linkedin Jobs Scraper to view jobs posted in Linkedin from Terminal

## Usage

```ruby

ruby linkedin_job_scraper.rb "Job Title"  "Place/Contry"

#Example

ruby linkedin_job_scraper.rb Ruby Germany

```
